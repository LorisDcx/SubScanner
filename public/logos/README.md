# Logos des marques

Ce dossier contient les logos des marques/abonnements détectés.

## Format recommandé
- Format : PNG avec fond transparent
- Taille : 48x48px minimum (idéalement 128x128px pour la netteté sur écrans HD)
- Nom du fichier : en minuscules, sans espaces (exemple: `netflix.png`, `amazon-prime.png`)

## Sources pour les logos
- Site officiel de la marque (section "Press Kit" ou "Brand Assets")
- [Brandfetch](https://brandfetch.com/)
- [Clearbit Logo API](https://clearbit.com/logo)
- Créer un placeholder avec les initiales de la marque en attendant

## Logo par défaut
Si aucun logo n'est disponible, le composant affichera les initiales de la marque dans un cercle coloré.
